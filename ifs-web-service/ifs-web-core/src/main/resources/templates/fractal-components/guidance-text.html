In IFS, the text input, text area and field set components can use grey guidance text. This is seen in the GDS standard components 'Text input' and 'Text area' where it is called 'hint text'. But in IFS hint text is used in more places to help the user complete form fields.

Example and code can be found here:
https://design-system.service.gov.uk/components/textarea/

Guidance on hint text here:
https://design-system.service.gov.uk/components/text-input/#using-hint-text
